Punycode

object Punycode

An RFC 3492 punycode decoder for converting ASCII to Unicode domain name labels. This is intended for use in Internationalized Domain Names (IDNs).

This class contains a Kotlin implementation of the pseudocode specified by RFC 3492. It includes direct translation of the pseudocode presented there.

Partner this class with Stringprep to implement IDNA RFC 3490.
