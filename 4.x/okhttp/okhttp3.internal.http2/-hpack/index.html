Read and write HPACK v10.

http://tools.ietf.org/html/draft-ietf-httpbis-header-compression-12

This implementation uses an array for the dynamic table and a list for indexed entries. Dynamic entries are added to the array, starting in the last position moving forward. When the array fills, it is doubled.
