ConnectPlan

class ConnectPlan(client: OkHttpClient, call: RealCall, chain: Interceptor.Chain, routePlanner: RealRoutePlanner, val route: Route, routes: List<Route>?, attempt: Int, tunnelRequest: Request?, connectionSpecIndex: Int, isTlsFallback: Boolean, connectionListener: ConnectionListener) : RoutePlanner.Plan, ExchangeCodec.Carrier

A single attempt to connect to a remote server, including these steps:

  • TCP handshake

  • Optional CONNECT tunnels. When using an HTTP proxy to reach an HTTPS server we must send a CONNECT request, and handle authorization challenges from the proxy.

  • Optional TLS handshake.

Each step may fail. If a retry is possible, a new instance is created with the next plan, which will be configured differently.
