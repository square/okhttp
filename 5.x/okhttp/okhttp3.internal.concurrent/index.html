fun formatDuration(ns: Long): String

Returns a duration in the nearest whole-number units like "999 µs" or " 1 s ". This rounds 0.5 units away from 0 and 0.499 towards 0. The smallest unit this returns is "µs"; the largest unit it returns is "s". For values in -499..499 this returns " 0 µs".
