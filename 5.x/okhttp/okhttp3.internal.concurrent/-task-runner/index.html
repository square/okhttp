TaskRunner

class TaskRunner(val backend: TaskRunner.Backend, logger: Logger = TaskRunner.logger) : Lockable

A set of worker threads that are shared among a set of task queues.

Use INSTANCE for a task runner that uses daemon threads. There is not currently a shared instance for non-daemon threads.

The task runner is also responsible for releasing held threads when the library is unloaded. This is for the benefit of container environments that implement code unloading.

Most applications should share a process-wide TaskRunner and use queues for per-client work.

fun awaitTaskToRun(): Task?

Returns an immediately-executable task for the calling thread to execute, sleeping as necessary until one is ready. If there are no ready queues, or if other threads have everything under control this will return null. If there is more than a single task ready to execute immediately this will start another thread to handle that work.
