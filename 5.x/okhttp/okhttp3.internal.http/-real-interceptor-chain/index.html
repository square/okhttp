RealInterceptorChain

class RealInterceptorChain(call: RealCall, interceptors: List<Interceptor>, index: Int, exchange: Exchange?, request: Request, connectTimeoutMillis: Int, readTimeoutMillis: Int, writeTimeoutMillis: Int, val authenticator: Authenticator, val cache: Cache?, val certificatePinner: CertificatePinner, val connectionPool: ConnectionPool, val cookieJar: CookieJar, val dns: Dns, val hostnameVerifier: HostnameVerifier, val proxy: Proxy?, val proxyAuthenticator: Authenticator, val proxySelector: ProxySelector, val retryOnConnectionFailure: Boolean, val socketFactory: SocketFactory, val sslSocketFactoryOrNull: SSLSocketFactory?, val x509TrustManagerOrNull: X509TrustManager?, val certificateChainCleaner: CertificateChainCleaner?) : Interceptor.Chain

A concrete interceptor chain that carries the entire interceptor chain: all application interceptors, the OkHttp core, all network interceptors, and finally the network caller.

If the chain is for an application interceptor then exchange must be null. Otherwise it is for a network interceptor and exchange must be non-null.
